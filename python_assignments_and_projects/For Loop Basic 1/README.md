# For Loop Basic I 🔁

A Python exercise from the **Python Stack Onsite** curriculum covering fundamental `for` loop patterns and algorithms.

## 📋 Assignment Overview

Create a Python file called `for_loop_basic1.py` that performs the following tasks:

---

## 🧩 Tasks

### 1. Basic
Print all integers from **0 to 150**.

### 2. Multiples of Five
Print all multiples of **5** from **5 to 1,000**.

### 3. Counting, the Dojo Way
Print integers **1 to 100** with the following rules:
- If divisible by **5**, print `"Coding"` instead
- If divisible by **10**, print `"Coding Dojo"` instead

### 4. Whoa. That Sucker's Huge
Add all **odd integers** from **0 to 500,000** and print the final sum.

### 5. Countdown by Fours
Print positive numbers starting at **2018**, counting **down by fours**.

### 6. Flexible Counter
Set three variables: `lowNum`, `highNum`, and `mult`.  
Starting at `lowNum` and going through `highNum`, print only the integers that are a multiple of `mult`.

**Example:** If `lowNum=2`, `highNum=9`, and `mult=3`, the loop should print:
```
3
6
9
```

---

## 🎯 Objectives

- Learn how to use basic `for` loop statements in Python
- Practice fundamental algorithms using loops
