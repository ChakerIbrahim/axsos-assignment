# 🐍 Python Stack — Organizing Files

## 📋 Assignment Overview

**Course:** Python Stack Onsite — Axsos Academy  
**Module:** Introduction & Setup  
**Assignment:** Organizing Files  

The goal of this assignment is to set up a well-structured folder system for all Python course files and projects, ensuring everything stays organized throughout the course.

---

## 🎯 Objectives

- Set up a folder structure to keep Python files organized
- Create a folder arrangement that aligns with course chapters
- Avoid spaces in all file and folder names

---

## 📁 Folder Structure

```
python_stack/
├── my_environments/
├── _python/                  ← (underscore prefix is required)
│   ├── python_fundamentals/
│   └── OOP/
└── django/
    ├── django_fundamentals/
    ├── django_orm/
    └── django_fullstack/
```

> ⚠️ **Note:** The underscore in `_python` is important and must not be omitted.

---

## ✅ What Was Done

The following folders were created inside `Documents/Axsos Course/`:

| Folder | Purpose |
|---|---|
| `python_stack/` | Root folder for the course |
| `-- _python/` | Python fundamentals & OOP work |
| `-- django/` | Django-related assignments |
| `-- my_environments/` | Virtual environments |
| `python_assignments_and_projects/` | Standalone projects |

---

## 💡 Key Rules

- No spaces in file or folder names — use underscores (`_`) instead
- Keep the `_python` folder name exactly as shown (underscore prefix matters)
- Structure should mirror the course chapter layout
